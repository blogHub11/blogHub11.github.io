简单数组：

1、最大子序和
  • 思路：
    • 如果数组中的所有元素都是负数的话，那么sum就是其中最大的负数的值，不存在连续子数组，也就是只包含一个元素
    • 如果sum大于0了，暂存值result，sum+=nums[i]，result和sum比较大小，这样保留的一直都是最大的值。
2、加一
  • 思路一：
    • 将数组中的元素转换为字符串，字符串转换为数字然后+1，再转换为数组。这样的问题隐患就是int的最大值溢出的问题。
  • 思路二：
    • 操作最末尾的元素，使最末尾的元素+1
    • 关键的操作就是 digits[i] %= 10;（余数为0， 说明进位了。）这样的做法是来检验末尾的元素是不是9+1，这样是存在进位的问题的。
    • 如果末尾元素不为0，直接返回数据。为0的话，需要继续循环处理。
    • 例如999+1，这样的话是需要数组扩容的，这样新创建一个数据，使新数组的第一个元素为1即可。
3、合并两个有序数组

  • 思路1：

    • System.arraycopy(int[] source, int sourceIndex, int[] target, int startIndex, int length);这个方法是实现数组的copy，将原数组copy到目标数组中。
    • 然后数组进行排序

  • 思路2（此处用到）：

    • 还是上面的方法，构建一个int[m+n]的数组，将数组1和数组2中要的子序都放进去，然后排序，在放回到数组1中

  • 思路3：

    • 双指针从后面开始排序。
